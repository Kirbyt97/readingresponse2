Pilgrim-Inroduction

The introduction in Pilgrim’s “HTML Book” begins by explaining what HTML is. HTML is made up of many different features and these features interact with other programs such as JavaScript. HTML5 expands on the previous model without making the users learn a whole new set of rules. These new features make using HTML much easier and allows for greater customization. It is easy to upgrade to this new model and all you have to do is change your doctype. This new model of HTML is already supported on popular browsers, listed below. The founder of the world wide web, Tim Berners-Lee, has condoned HTML5 and plans to help it develop further.

  • Firefox
  • Safari
  • Chrome
  • Mobile Browsers

    What is a MIME>
    A MIME is a header that lets the browser know how to display the page. Things like images and JavaScript and countless others also have their own MIME type. Everything that is displayed on the web, for the most part, has a MIME type.

    The Development of a Tag
    Pilgrim displays a now twenty-four-year-old email chain between several important developers. This chain marks the beginning of the “IMG” tag. This tag is used in almost all modern websites and is very important. Developers collaborate and share their ideas to come up with the best version of the tag. However, it was Marc Andreessen of Mosaic who shipped the code. This is why it is called IMG, not ICON or INCLUDE.

    The Difference with XHTML and MIMES
    HTML is a web designer that allows for simple mistakes to be “forgiven.” This lead to the vast majority of all HTML web pages to have some sort of an error in them. XML set out to fix this problem. In this new model, it was not possible to fail on the first error for not being well formed. Most people did not like this new feature because it displayed constant errors to the user due to most existing pages having these errors. However, many people preferred XHTML syntax so using it with an HTML MIME type became popular. This caused HTML to develop new syntax and forms which lead to the creation of HTML5.

    A VT-Shaped Student

    This is a brief summary of what a VT-Shaped Student is as outline on the Virginia Tech website and relating articles. Virginia Tech takes great pride in their student’s achievements. In an attempt to encourage students to better themselves even further Tech has created the notion of a VT-Shaped Student.

    The T
    There are two parts to the T:

  • Horizontal
  • Vertical
    • Horizontal
    Represents the strength a student as within their chosen discipline. Tech students are expected to be the best at what they do in their given fields.
    Vertical
    The vertical portion represents being an interdisciplinary student. It is required that students take courses outside of their chosen fields in order to expand their knowledge.

    The V
    The V-This represents students learning outside of the class room. Tech students are also expected to be active members in their community and live out the school motto of Ut Prosim. At Tech there are many opportunities to learn outside of class such as:

  • Living-Learning Communities
  • Study Abroad
  • Undergraduate Research
